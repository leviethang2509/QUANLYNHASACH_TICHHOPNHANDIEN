# QLNhaSach_BaoCao

Thu muc nay dung de quan ly toan bo qua trinh xay dung bao cao cho du an QLNhaSach.

## Quy tac quan ly

1. Tat ca file sinh ra trong qua trinh viet bao cao phai nam trong thu muc nay.
2. Khong ghi de file goc cua du an neu chua co yeu cau ro rang.
3. Moi quy trinh can co lich su trong `13_history` hoac nhat ky trong `11_logs`.
4. File ket qua cuoi cung dat trong:
   - `09_exports/docx`
   - `09_exports/pdf`
5. Cac phan trung gian duoc tach rieng theo dung thu muc de de kiem tra lai.

## Cau truc chinh

- `00_inputs`: ban sao ke hoach, template Word va cac tai lieu dau vao.
- `01_template_markdown`: noi chuyen template Word sang Markdown.
- `02_source_analysis`: phan tich source code, database, workflow va bao mat.
- `03_knowledge_base`: tri thuc tong hop cua du an.
- `04_generated_sections`: cac muc noi dung sinh ra cho bao cao.
- `05_uml`: PlantUML, Draw.io va anh so do.
- `06_report_markdown`: cac chuong Markdown va ban tong hop.
- `07_tables`: bang yeu cau, database, API, test case.
- `08_testing`: noi dung kiem thu.
- `09_exports`: DOCX/PDF cuoi.
- `10_validation`: ket qua kiem tra chat luong.
- `11_logs`: nhat ky thuc thi.
- `12_archive`: ban luu tru.
- `13_history`: lich su tung quy trinh.
